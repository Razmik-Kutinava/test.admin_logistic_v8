# ⚠️ СРОЧНО: Применить миграцию базы данных

## Проблема
```
The table 'public.drivers' does not exist in the current database.
```

## Решение за 2 минуты:

### 1. Откройте файл `backend/APPLY_MIGRATION.sql`

### 2. Скопируйте ВСЁ содержимое (Ctrl+A, Ctrl+C)

### 3. Откройте Supabase Dashboard:
   - https://app.supabase.com
   - Выберите проект
   - SQL Editor (левое меню)
   - New Query

### 4. Вставьте SQL и нажмите Run (Ctrl+Enter)

### 5. Готово! Перезапустите backend
